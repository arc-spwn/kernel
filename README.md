# Arc Kernel
Arc's kernel written in Rust

## Requirements
- Linux *At the moment.*
- Rustup
- Rust's nightly toolchain. *Can be installed by `make setup-env`.*
- Cargo bootimage *Can be installed by `make setup-env`.*

## Building
- If you don't have the last two requirements already installed.
    - Run `make setup-env`.
- Run `make compile`.
